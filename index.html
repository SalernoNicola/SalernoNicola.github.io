A machine learning-based methodology for the classification of brain tumors

University of Salerno

This work was carried out during the Magistral thesis whose purpose is to predict the types of tumor that a patient is affected given magnetic resonance images, optimizing the traditional Machine Learning techniques present in the literature.

The study covered various phases ranging from the removal of non-brain tissue, named skull-stripping , to the actual tumor segmentation. Next, a convolutional neural network for image classification was implemented initially. To optimize the state of the art, additional features were obtained from the images using the Gray Level Co-Ocurrence Matrix and other information from the patient's medical record To optimize the state of the art, additional features were obtained from the images using the Gray Level Co-Ocurrence Matrix and other information from the patient's medical record. These features were subjected to a Discovery Functionality of Relaxed Functional Dependencies algorithm to extract the most important features that were later linked in the classification process.
